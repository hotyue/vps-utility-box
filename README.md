# 杂货铺 (vps-utility-box) v2.0.0 🏪

一键交互式部署工具，根据您的需求定制 VPS 维护方案。

## 🚀 一键部署

在您的 VPS (Debian 12+) 上直接运行以下指令：

```bash
curl -sSO https://raw.githubusercontent.com/hotyue/vps-utility-box/main/deploy.sh && bash deploy.sh
```
